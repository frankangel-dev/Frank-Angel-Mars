# Frank Angel's Portfolio

A personal portfolio website built as part of the Intro to Programming course with **Code the Dream**. This project showcases my skills, experience, and projects while demonstrating foundational web development concepts.

## Features

- **Dynamic Projects:** Automatically fetches and displays repositories from GitHub using the GitHub REST API.
- **Interactive Message Board:** A custom-built form allowing users to leave, edit, and remove messages with real-time feedback.
- **Dark/Light Mode:** A smooth theme toggle to enhance user experience and accessibility.
- **Responsive Design:** A fully mobile-friendly layout featuring a functional hamburger menu and flexible layouts.
- **Form Validation:** Client-side validation for email formats and required fields.

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/frankangel-dev/Frank-Angel-Mars.git
   ```
2. Open `index.html` in any modern web browser.

## Technologies Used

- **HTML:** Semantic structure.
- **CSS:** Custom styling using CSS, CSS Variables for themes, and Flexbox for responsive design.
- **JavaScript:** DOM manipulation, event handling, and asynchronous API calls (`fetch`, `async/await`).
- **GitHub API:** Integration to dynamically populate the projects section.
